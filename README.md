# Incremental

A browser-based incremental/idle game built with HTML, CSS, and vanilla JavaScript. This project is fully deployed using GitHub Pages with an automated CI/CD pipeline powered by GitHub Actions. [Try the live demo](https://klykinsjobs.github.io/Incremental/).

## Getting started
- Clone the repository
- Open index.html with your preferred web browser
