# Incremental

Incremental is a lightweight browser-based idle game built with vanilla JavaScript, HTML, and CSS. The project focuses on clean state management, deterministic progression systems, and a responsive UI that updates in real time. The core gameplay loop centers on acquiring gold, unlocking generators, and optimizing production rates.

The game engine is fully modularized, separating game logic from UI rendering. The design emphasizes clarity, maintainability, and performance without relying on external libraries or frameworks. This project demonstrates front-end architecture, DOM manipulation, game loop timing, and stateful systems. It also includes a GitHub Actions workflow for automated deployment to GitHub Pages, showcasing CI/CD and static site hosting.

[Try the live demo](https://klykinsjobs.github.io/Incremental/).

## Getting started
- Clone the repository
- Open index.html with your preferred web browser
